[中译] Scroll 基础教程

April 9, 1983 — Scroll 是 Markdown 的可扩展替代品。

与 Markdown 一样，Scroll 是一种可编译为 HTML 的纯文本语言。

滚动是最小的，可以帮助您将想法简化为本质。

Scroll 是可扩展的，可以帮助您使用和构建新的迷你语言来充分发展您的想法。

Scroll 于 2021 年首次亮相，目前版本为 72，大约每 2 周发布 1 次。目前还没有官方规范，但创建规范已在路线图上。

本教程将引导您了解 Scroll 中的大多数关键字，并教您如何构建自己的 Scroll 关键字。

一个 Scroll 文档

Scroll 文档（或“程序”）是节点列表。每个节点都是一条线，每条线都是一个节点。

如果在一行的开头放置一个空格，则该行将成为前一行的子行。您之前可能在 Python 等语言中见过这种缩进技巧。但滚动将其推到了极限。

如果你掌握了缩进技巧，你就掌握了滚动。但我们有点超前了，让我们从基础开始吧。

链接

Scroll有着不同的超链接方式。您无需将链接与内容混合在一起，而是将链接在文本之后与您希望链接匹配的文本一起放置。例如：

维基百科的链接 https://wikipedia.org Wikipedia

列

您可以使用startColumns [maxNumberOfColumns]关键字来启动列流，并使用endColumns来结束列流。如果您不希望某个部分跨列分隔，请不要在行之间放置换行符。换行符将清除部分。

使用自定义主题

默认情况下，Scroll 发出没有主题的 HTML。要使用任何主题（包括默认的Gazette主题），用户只需使用主题提供的关键字，例如gazetteCss。

专家：添加你自己的解析器

Scroll 基于这样的理论：语言应该适应领域，而不是相反。所以 Scroll 具有内置的可扩展性。

注意：当前仅使用npm包支持自定义解析器。网页编辑器目前不支持自定义关键字。

您可以使用*Parser在 Scroll 文档中定义自己的解析器。这些解析器是用Grammar语言编写的。

下面是一个简单的例子，它通过使p与*的工作方式相同来扩展 Scroll：

pParser extends thoughtParser crux p p We can then make paragraphs using `p`.
